# sql-script-generator
一个功能强大的 SQL 脚本生成器工具，帮助开发者快速创建 SQL 模板文件。
[https://github.com/lov3smu/sql-script-generator](https://github.com/lov3smu/sql-script-generator)

# ✨ 功能特性
## 🚀 核心功能
**快速 SQL 生成：** 通过图形界面快速生成各种 SQL 语句

**模板管理：** 内置丰富的 SQL 模板，支持自定义模板

**多数据库支持：** MySQL、PostgreSQL、SQLite、Oracle、SQL Server

**语法高亮：** 实时 SQL 语法检查和着色

**代码格式化：** 自动格式化生成的 SQL 代码

## 📁 模板类型
**DDL 模板：** CREATE TABLE、ALTER TABLE、DROP TABLE

**DML 模板：** INSERT、UPDATE、DELETE

**DQL 模板：** SELECT

# 📦 安装
## 下载安装
从 [Releases 页面](https://github.com/lov3smu/sql-script-generator/releases)下载对应平台的安装包：

| 平台 | 下载链接 | 说明 |
|:---|:---|:---|
| 🪟 Windows | SQL-Script-Generator-Setup-*.exe | 标准安装程序 |
| 🍎 macOS | SQL-Script-Generator-*.dmg | 磁盘镜像文件 |
| 🐧 Linux | SQL-Script-Generator-*.AppImage | 通用可执行文件 |
	

## 系统要求
**Windows:** Windows 10 或更高版本

**macOS:** macOS 10.15 (Catalina) 或更高版本

**Linux:** Ubuntu 18.04+ 或兼容发行版

**内存:** 至少 2GB RAM

**磁盘空间:** 至少 200MB 可用空间

# 🚀 快速开始

启动 SQL Script Generator

选择数据库类型 (如 MySQL)

选择模板类型 (如 CREATE TABLE)

点击"生成"按钮
